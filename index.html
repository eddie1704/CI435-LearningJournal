Week 7: Flexible media

Published on November 17, 2022

In this tutorial I have experimented with using differant ways to make images and videos on a webpage flexible to fit on a responsive design for a website.

Today's work will be applied to a differant HTML document that can be found here: Flexible media examples

I have learnt how to:

  1. Use a Clipped full-size Image to use a page banner - the image keeps it's size on the page regardless of the brower window.
  2. Use a Flexible Image to use as a page banner - the image gets smaller to maintain the width of the browser window.
  3. Use a Flexible Image with Text Wrap to place next to relevant text on the page - the image gets smaller to maintain equal space with the text on the page.
  4. Use a Flexible Video Container to embed a video on a page - the video gets smaller to maintain the width of the browser window.

Week 6: Making the Learning Journal page Responsive

Published on November 10, 2022

In this tutorial I have learnt how to make my Learning Journal into a responsive website so it is easy to view and navigate on various devices including mobile phones, tablets, laptops, and desktop computers.

I have planned how I want to make my Learning Journal responsive by making some Wireframes for the Mobile and Desktop layouts for my webpage. Click to view via OneDrive: Desktop Mobile

I have also made all the images on this page compressed JPGs that are atleast 852x480 no larger, so that the website loads quickly on devices with a slow or limited network connection. Like before you can still click on them to open the original PNG image.

Today I have made the following changes:

  1. Adjusted some parts of my HTML (The TimeTable is now at the top of the page, under the Head. )
  2. I have setup a responsive grid in the CSS for a 1 column, 2 column and 3 column layouts.
  3. I have chnaged the navigation menu at the top of the page using media queries so that it styles as buttons that are horizontal for devices with wide screens.
  4. I have used the display:none tag to hide the timetable on a mobile device in portrait orientation to make the weekly posts easier to see.
  5. I have used the CSS rule img { max-width: 100%; } so that all the images scale automatically to the width of the parent element. This then makes the images on the webpage responsive and fit nicely on the device's screen.

Week 5: CSS - Styling Text

Published on November 3, 2022

In this tutorial I have learnt how to style the text on the body of the webpage. This is to enhance the appearance and readability of the webpages as text is considered the most important content on a webpage.

Today I learnt how to:

  1. Specify a font-family with a fallback - this is to ensure that there is a common font to fall back to if the user does not have the preferred font on their device.
  2. Use web fonts using Google's free font library to have more customised font families on the webpage. (I have set the font Ubuntu for the headings!)
  3. Use the tags: line-height, margin, text-indent, dropcap and text-align to style the paragraphs on my page. This is becuase it makes more space between the lines of text, enhancing legibility and the overwall appearance of the website. (You can see where i've used the dropcap tag at the begining of this post.)
  4. Use float and clear properties in the style sheet to lay out images on the webpage so that content wraps around them. float is used to indicate the start of it and clear is used to end the float.
  5. I have also added top to bottom gradient to the Post headers (div id="post") as I prefer gradient over flat clolurs in User Interfaces and have made the blue text a little lighter coloured for readability.
  6. I have also learnt in my own time how to use the Target Attribute element on the href tag in order to make external links open in a new tab in the device's web browser. I have applied this change to all external links on this page!

For Example: a href="https://www.freecodecamp.org/" target="_blank"
